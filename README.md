# Coding Assignment-1 Solutions

## Problem 1: Cyclic Substring Maximum Sum
- Uses sliding window technique
- Handles circular substring using string doubling
- Time Complexity: O(N)

## Example
Input:
abca

Output:
6

## Problem 2: Array Transformation Cost Minimization
- Uses median for optimal solution
- Checks divisibility condition
- Time Complexity: O(N log N)

## Example
Input:
5
2 4 6 8 10
2

Output:
6
